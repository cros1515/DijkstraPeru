Cómo ejecutar el proyecto
1. Crea una carpeta, puede ser "tu_repositorio".
2. En dicha carpeta abre la terminal de git.
3. Una vez en la terminal Inicializa y Clona el repositorio:
   1. git init
   2. git clone https://github.com/cros1515/DijkstraPeru.git
4. Una vez hecho todo esto el proyecto ya debería estar en tu carpeta, solo quedaría abrir el archivo index.html en tu navegador.

Recomendación: usar Live Server en VS Code para evitar problemas de carga.
